1. The Big Picture: When & Why You Need the Delta Method

The core problem: In experimentation, we need variance to build confidence intervals and determine statistical significance. But calculating variance correctly depends on whether our data points are independent.

3. Clustering — Why It Matters

Independent (between clusters)

Account A and Account B are independent — assigned separately to control/treatment.

Each account = one randomization unit = one cluster.

Correlated (within a cluster)

Requests within Account A share network conditions, user behavior, etc.

Request a1 and a2 are not independent — they're correlated.

Key insight: We can't compute variance directly on the request-level data because requests aren't i.i.d. But accounts are i.i.d. — so we rewrite the metric in terms of accounts.

4. Linear vs. Nonlinear — Why Ratios Are Tricky

LINEAR Simple Mean

When randomization unit = analysis unit:

avg = (x₁ + x₂ + … + xₙ) / n = (1/n)·x₁ + (1/n)·x₂ + …

Multiply by constant, add up → linear!

Standard variance formula works.

NONLINEAR Ratio of Means

When we rewrite for a different randomization unit:

ratio = mean_numerator / mean_denominator

Division of two random variables → nonlinear!

Can't naively compute variance. Need the Delta Method.

What goes wrong without the Delta Method?
  • Variance too small → false confidence → Type I errors (false positives)
  • Variance too large → overcautious → Type II errors (miss real effects)

Step-by-Step with Numbers

1 Aggregate to randomization unit (account)
Account AAccount B
Numerator (sum of latency)10 + 20 = 30 ms5 + 20 + 30 = 55 ms
Denominator (count of requests)23

Now each observation is one account — and accounts are i.i.d.!

2 Compute sample statistics (with Bessel's correction, n−1)
StatisticFormulaValue
μ_S (mean numerator)(30 + 55) / 242.5 ms
μ_N (mean denominator)(2 + 3) / 22.5 requests
σ²_S (var numerator)((30−42.5)² + (55−42.5)²) / 1312.5 ms²
σ²_N (var denominator)((2−2.5)² + (3−2.5)²) / 10.5 req²
σ_SN (covariance)((30−42.5)(2−2.5) + (55−42.5)(3−2.5)) / 112.5
3 Plug into the Delta Method

K = 2 accounts

Var(Ý) = (1 / (K · μ_N²)) × (σ²_S − 2·(μ_S/μ_N)·σ_SN + (μ_S/μ_N)²·σ²_N)
= (1 / (2 · 2.5²)) × (312.5 − 2·(42.5/2.5)·12.5 + (42.5/2.5)²·0.5)
= (1 / 12.5) × (312.5 − 425 + 144.5) = (1/12.5) × 32 = 2.56
